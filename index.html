WANDERNG THE WORLD

Wandering the World is a website for all of your travel needs. As the winner of Code First Girl's final project challenge, we were tasked to create an interactive website that included a taste of what we liked to do in our spare time. And who doesn't love a holiday?

This website uses three core web technologies - HTML, CSS, and Javascript. Bootstrap is used to create the fundamental parts within the landing page's wireframe, such as a carousel of locations, an offcanvas navbar, and destination cards to explore.

It also includes an interactive map finder, contact form, destination quizzes and fun facts!
