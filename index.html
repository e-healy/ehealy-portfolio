WANDERNG THE WORLD

Wandering the World is a website for all of your travel needs. As the winner of Code First Girl's final project challenge, we were tasked to create an interactive website that included a taste of what we liked to do in our spare time. And who doesn't love a holiday?

This website uses three core web technologies - HTML, CSS, and Javascript. Bootstrap is used to create the fundamental parts within the landing page's wireframe, such as a carousel of locations, an offcanvas navbar, and destination cards to explore.

It also includes an interactive map finder, contact form, destination quizzes and fun facts!

AUTO WHIZ

Need a car? Don't worry, Auto Whiz has you covered. With a database of around 100 cars to choose from, Auto Whiz assists you in finding your dream vehicle, letting the user input their ideal car model, make and price.

Once the user finds their ideal car, they can favourite it which will add it to their car wishlist. From there, they can decide whether they want to purchase or pick another car!

This website was built using HTML, CSS. I loved building this project; it required me to push my javascript skills to the next level and develop a solution-focused mindset. As a result, this project was the winner of Code First Girl's Introduction to Javascript final project challenge.

GUESS THE NUMBER

In this project, I used Java to create a simple guess the number game. The game is simple: pick the right number, you win.

Although in this game, you make the rules. The console will ask you to guess a number from 1 to whatever range you would like to set it as, you will also input how many attempts you would like to complete it in.

This was my first Java project, and I thoroughly enjoyed building it. I learned basic OOP principles, such as inheritance, abstraction and encapsulation to build this project. I also learned about constructors, as well as getters/accessors used to facilitate my code.
